Webcms3 XSS Vulnerability

Webcms3 XSS Vulnerability

With permission from the owner of the website, Jas, I explored an XSS vulnerability within webcms3. The vulnerability was quite hard to exploit as code could only be injected into the settings of your own profile page. This turned a normal XSS into something else - Self XSS. Self XSS is a combination of social engineering and xss. For the details on how I performed the Self XSS, check out the above powerpoint presentation that I gave for one of my university courses, COMP6441 - Security Engineering
